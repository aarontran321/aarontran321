## 🚀 Featured Projects

### BAAM
*Solana • Rust • TypeScript • Swift • MongoDB • Vultr | JAMHacks 2026*

A social betting platform for close friend groups, shipping multiple surfaces simultaneously.

* Built a Solana smart contract backend powering on-chain wagers with full financial settlement
* Shipped a native iMessage plugin in Swift, a Discord bot for bet notifications, and a centralized web app
* Leveraged MongoDB Atlas for data persistence and Vultr for reliable backend hosting
* Collaborated cross-functionally with a team to design and ship the entire stack within the hackathon timeline

### Retermina
*Tauri • Rust • React • TypeScript | 2026 - Present*

A cross-platform desktop terminal workspace engineered for power users and developers.

* Built on Tauri v2 with a Rust backend driving native PTY shells—fully local, zero cloud dependency, no subscriptions
* Engineered a draggable multi-panel layout with react-grid-layout: split terminals, syntax-highlighted editor, file explorer, live project-wide git diff, localhost tracker, and native preview
* Embedded a Claude Code CLI with per-project token tracking and Iris, a context-aware command bar that surfaces git, npm, and shell macros based on live repository state
* Implemented five structural theme engines with portable presets for instant full-application re-skinning
